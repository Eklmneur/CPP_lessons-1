Шаблон класса -- проект, по которому создаются классы.
Они нужны, чтобы не создавать отдельно каждый класс.
1)
Могут быть типы (typename T), могут переменные (char c).
В переменные можно отправлять только константные значения.
2)
Что значит рекурсивно? Если это значит возможность создания
чего-то типа pair<vector<int>, pair<int, set<double>>>.
3)
Сколько угодно, только конечное число.
4)
Можно, всё по аналогии с аргументами функций.
5)
Можно специализировать некоторые аргументы шаблонов.
Пусть есть template <typename q, typename w, typename e>
class newclass
Специализация по второму аргументу выглядит так:
template <typename q, typename e>
class newclass<q, int, e>
6)
Передать можно. Вопрос в том, что если это объекты
двух разных специализаций шаблона, то 
функция может быть разной, может даже 
отсутствовать для различных специализаций.
Т.е., всё зависит от того, есть ли эта функция
в специализациях.
7)
Только первый, так как считаются что различные
специализации шаблона есть совершенно различные
классы, не имеющие ничего общего.
